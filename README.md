# PHP Version of REYD Web Application

This folder contains a PHP implementation of the same website structure:
- Home
- Products
- Services
- Contact section

## Run Locally

From the `php-app` directory:

```bash
php -S localhost:8080
```

Then open `http://localhost:8080`.

## Contact

The webpage now provides direct contact options (phone and email) instead of an on-page contact form.

If you later re-enable a server-side contact form, add a backend handler and configure mail transport based on your hosting provider limitations.
